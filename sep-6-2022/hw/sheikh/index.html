"Some Shortcuts"
  1. Alt+tab = to back window
  2. Alt+tab(continue) = to back & back window
  3. Alt+F4 = to close 'Vs code'
  4. Ctrl+W = to close 'get started' in VS code
  5. Ctrl+page up = to change window in 'VS code'
  6. Ctrl+B = to close file explorer tab in 'VS code'
  7. Ctrl+J = to open command promt in 'VS code'
  8. Alt+shift+down arrow = to duplicate line 'VS code'
  9. Alt+left click on mouse = to duplicate cursor 'VS code'
  10. Ctrl+alt+down arrow= to duplicate cursor in 'VS code'
  11. Shift+right arrow/left arrow= to duplicate cursor in 'VS code'
  12. (After select word) Ctrl+F2= to select same word continuing after line in 'VS code'
